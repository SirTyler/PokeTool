# PokéTool
Streamer assistant for Pokémon Games.

Shiny Encounters have Hotkeys of:    
Increase - PLUS    
Decrease - MINUS    
Reset    - DIVIDE    

These can be changed in the created encounters.txt file    
*uses java AWT Keystrokes*
